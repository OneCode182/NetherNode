# Context Loading Protocol

## Levels

- Level 0: root `AGENTS.md`, `.agents/AGENTS.md`, `.agents/env.json`.
- Level 1: active workflow and task.
- Level 2: relevant project/architecture/knowledge index.
- Level 3: specific memory file.
- Level 4: implementation file excerpts.

## Rule

Load indexes before broad reads. Use Graphify only after checking Markdown authority docs.
